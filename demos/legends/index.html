Heading and legends in AF

This is a test page to see how headings inside legends are exposed to screen-readers.

Both the following fieldsets have a h1 inside the legend.

The first example (drinks) has a css style of "overflow hidden" added to the legend to fix a bug in Safari where margins on contained headings were not honoured.

This style exists as part of Gov Elements library.

The second example (cakes) does not have this style and acts as a control.
